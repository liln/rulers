# Rulers

A Rack-based web framework with ninja powers.

## Installation

Add this line to your application's Gemfile:

    gem 'rulers'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rulers

## Usage

Use with caution. Ninjas may appear.
